# Техническое задание
## Проект: TOOLBOX — Универсальный арсенал

**Версия:** 1.0  
**Дата:** Апрель 2026  
**Статус:** Актуально

---

## 1. Общее описание

TOOLBOX — одностраничное веб-приложение (SPA), предоставляющее набор утилит для разработчиков и дизайнеров. Работает полностью в браузере без серверной части и регистрации. Все операции выполняются на клиенте (client-side), за исключением загрузчика медиа, обращающегося к внешним cobalt-инстансам.

### 1.1 Цели проекта

- Объединить часто используемые инструменты разработчика в одном месте
- Обеспечить работу без интернета (offline-ready, static HTML)
- Не собирать данные пользователей, не показывать рекламу
- Поддерживать русскоязычный интерфейс

### 1.2 Технический стек

| Компонент | Технология |
|-----------|-----------|
| Фронтенд | Vanilla HTML/CSS/JS (без фреймворков) |
| Шрифты | Plus Jakarta Sans, Unbounded, JetBrains Mono (Google Fonts) |
| Внешние библиотеки | qrcodejs (QR-коды) |
| Хранилище | localStorage (браузер) |
| Деплой | Статический HTML-файл, не требует сервера |

## 2. Интерфейс и навигация

### 2.1 Общая структура

Приложение состоит из двух зон:

- **Сайдбар (240px)** — фиксированная боковая панель с навигацией, логотипом и переключателем темы
- **Основной контент** — область с секциями инструментов, отображается по одной секции за раз

Переключение между секциями — мгновенное (без перезагрузки страницы), с анимацией `fadeIn`.

### 2.2 Навигационное меню (сайдбар)

Разделы сгруппированы по категориям:

**Текст:**
- 📝 Текст
- 🔀 Diff
- 📄 Markdown

**Генерация:**
- 🎨 CSS
- 🎲 Генераторы
- 🌈 Цвет
- 📱 QR-код

**Утилиты:**
- 🔄 Конвертер
- 🔐 Encode
- 📏 Единицы
- ⬇️ Загрузчик
- 💾 Записи

### 2.3 Темы оформления

Поддерживаются две темы:

- **Тёмная (по умолчанию)** — фон `#0d0d14`, акцент `#6ee7b7` (зелёный)
- **Светлая** — фон `#f4f4f8`, акцент `#0a7c56` (тёмно-зелёный)

Выбранная тема сохраняется в `localStorage` и восстанавливается при следующем открытии. Переключатель расположен в сайдбаре.

## 3. Модули (инструменты)

### 3.1 Главная страница

**Описание:** Стартовый экран с сеткой карточек всех инструментов.

**Требования:**
- Отображать все 12 инструментов в виде адаптивной сетки
- Каждая карточка содержит: иконку, название, краткое описание
- Клик по карточке — переход в соответствующий раздел
- Сетка: `repeat(auto-fill, minmax(260px, 1fr))`

---

### 3.2 Текстовые инструменты (`/text`)

**Описание:** Набор операций над текстом.

#### 3.2.1 Анализ текста
- Подсчёт символов (с пробелами и без)
- Подсчёт слов, строк, предложений
- Оценка времени чтения (200 слов/мин)

#### 3.2.2 Смена регистра
- ВЕРХНИЙ, нижний, Title Case, Sentence case

#### 3.2.3 Поиск и замена
- Простая замена подстроки
- Замена с поддержкой RegExp (флаг `g`)

#### 3.2.4 Прочие операции
- Разворот строки
- Trim (убрать пробелы по краям каждой строки)
- Сортировка строк (алфавит)
- Удаление дубликатов строк

#### 3.2.5 Slug / транслитерация
- Транслитерация кириллицы → латиница
- Генерация URL-slug (строчные, дефисы)

---

### 3.3 Diff — сравнение текстов (`/diff`)

**Описание:** Построчное сравнение двух текстов с визуальной подсветкой.

**Требования:**
- Два поля ввода: «Текст A (оригинал)» и «Текст B (изменённый)»
- Алгоритм: LCS (Longest Common Subsequence), реализация на JS
- Отображение: двухколоночная таблица с нумерацией строк
- Подсветка: зелёный фон — добавленные строки, красный — удалённые
- Статистика: количество добавленных, удалённых, совпавших строк
- Кнопки: «Сравнить», «Поменять местами», «Очистить»

---

### 3.4 Markdown редактор (`/md`)

**Описание:** Split-screen редактор Markdown с живым превью.

**Требования:**
- Разделённый вид: редактор (левая часть) + превью (правая часть)
- Высота панели: 600px на десктопе
- Тулбар с кнопками форматирования: Bold, Italic, Code, Code Block, H1–H3, Quote, List, Numbered List, Link, Image, HR, Table, Task
- Поддерживаемый синтаксис: заголовки, жирный, курсив, inline-код, блок кода, цитаты, списки, таблицы, task-листы, горизонтальная черта, ссылки, изображения
- Статистика слов в реальном времени
- Экспорт: `.md` файл, `.html` файл, Copy HTML

---

### 3.5 CSS генератор (`/css`)

**Описание:** Визуальный конструктор CSS-свойств с live-превью и готовым кодом.

**Подразделы (вкладки):**

| Вкладка | Параметры |
|---------|-----------|
| 🌈 Градиент | Тип (linear/radial/conic), угол, 2+ цвета с позициями |
| 🌑 Box Shadow | X, Y, blur, spread, opacity, цвет, inset |
| ✏️ Text Shadow | X, Y, blur, цвет, размер шрифта, текст превью |
| ⬛ Border Radius | Единый, индивидуальный, blob (случайный) |
| 🔮 Glassmorphism | Opacity, blur, saturate, border, radius |
| 🔷 Neumorphism | Фон, размер, blur, radius, inset |
| 🎬 Animation | Тип (8 вариантов), duration, timing, iteration |
| 🖼️ Filter | blur, brightness, contrast, saturate, hue-rotate, sepia, grayscale, invert |

**Требования:**
- Каждая вкладка содержит контролы (range/color), live-превью элемента и блок с готовым CSS-кодом
- Кнопка «COPY» для копирования CSS в буфер обмена
- Анимации воспроизводятся по кнопке «▶ Запустить»

---

### 3.6 Конвертер (`/convert`)

**Описание:** Конвертация между текстовыми форматами данных.

**Подразделы:**

- **JSON → CSV** и **CSV → JSON**: поля ввода/вывода, кнопка конвертации, Copy
- **Markdown → HTML**: ввод MD, вывод HTML, кнопки «→ HTML» и «Скачать»
- **JSON форматтер**: форматировать (2 пробела / 4 пробела), минифицировать
- **Загрузка файла**: поддержка `.json`, `.csv`, `.md`

---

### 3.7 Загрузчик медиа (`/download`)

#### 3.7.1 Прямое скачивание (cobalt API)

**Описание:** Скачивание медиа напрямую через браузер без установки ПО.

**Требования:**
- Поле ввода URL с кнопкой «Скачать»
- Поддерживаемые сайты: YouTube, TikTok, Twitter/X, Instagram, Vimeo, SoundCloud, Twitch, Reddit, Bilibili, Rutube, VK, Pinterest, Dailymotion, Tumblr и 20+ других
- Режимы: Видео + аудио / Только аудио / Видео без звука
- Выбор качества: Max / 4K / 2K / 1080p / 720p / 480p / 360p
- Выбор формата аудио: best / mp3 / ogg / wav / opus
- Логика: последовательный перебор cobalt-инстансов до первого успешного ответа
- Эндпоинты: пробуется `/api/json` (cobalt v10), затем `/` (cobalt v9)
- Скачивание через `fetch → Blob → createObjectURL` (принудительный save dialog)
- Fallback: прямая ссылка через `<a target="_blank">`
- Статус-индикатор: loading / success / picker (выбор файла) / error
- Бейдж активного инстанса с hostname в сайдбаре загрузчика

**Список cobalt-инстансов (проверяются при загрузке):**
```
https://cobalt.api.vvf.me
https://cobalt.flurl.dev
https://cobalt.urdnot.gay
https://cobalt.bato.sh
https://cbl.dom.taxi
https://cobalt.nadeko.net
https://cobalt.drgns.space
```

#### 3.7.2 Генератор команд yt-dlp — Видео

Параметры: URL, макс. качество (240p–8K), видеокодек, контейнер, аудиокодек, шаблон имени файла, папка вывода, ограничение скорости, таймкод начала/конца, флаги: плейлист / субтитры / обложка / главы / SponsorBlock / cookies.

#### 3.7.3 Генератор команд yt-dlp — Аудио

Параметры: URL, кодек (16 вариантов включая FLAC, ALAC, PCM), качество, частота дискретизации, папка, флаги.

#### 3.7.4 Генератор команд FFmpeg

Вкладки: Конвертация, Сжатие (с 2-pass), Аудио, Обрезка (copy/re-encode), GIF/WebP/APNG.

#### 3.7.5 Генератор команд spotdl

Параметры: URL, формат, битрейт, тексты песен, M3U-плейлист, многопоточность.

#### 3.7.6 Генератор команд gallery-dl

Параметры: URL, папка, лимит, ZIP, метаданные, конвертация ugoira.

## 4. Нефункциональные требования

### 4.1 Производительность
- Первый экран должен загружаться менее чем за 2 секунды на стандартном соединении
- Все вычисления (кроме загрузки медиа) — синхронные, без лагов UI

### 4.2 Совместимость
- Поддерживаемые браузеры: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- Мобильные устройства: адаптивная вёрстка, гамбургер-меню при ширине < 900px
- На мобильных: `.grid-2`, `.grid-3` переключаются в 1 колонку; MD-редактор — вертикальный split

### 4.3 Безопасность
- Никакие данные пользователя не передаются на сторонние серверы (кроме URL при использовании cobalt)
- localStorage хранится только локально в браузере

### 4.4 UX / визуальный стиль
- Шрифт заголовков: Unbounded (900/700)
- Шрифт интерфейса: Plus Jakarta Sans (400/500/600/700)
- Моноширинный: JetBrains Mono (400/700)
- Скругления: 14px (карточки), 8px (инпуты/кнопки)
- Анимации: `fadeIn 0.25s ease`, hover-transitions 0.18–0.22s
- Toast-уведомления: снизу справа, 2.5 сек, spring-анимация

## 5. Архитектура файла

Проект реализован как единый `index.html` файл:

```
index.html
├── <head>
│   ├── Подключение шрифтов (Google Fonts)

│   └── <style> — весь CSS (~900 строк)
└── <body>
    ├── #toast — глобальные уведомления
    ├── .hamburger — кнопка мобильного меню
    ├── .sidebar — навигация
    ├── <main> — секции инструментов
    │   ├── #home
    │   ├── #text
    │   ├── #diff
    │   ├── #md
    │   ├── #css
    │   ├── #convert

    │   ├── #encode
    │   ├── #gen
    │   ├── #color
    │   ├── #qr
    │   ├── #units
```

## 8. Ограничения и известные особенности

- **Cobalt инстансы** — публичные серверы третьих лиц, могут быть недоступны; приложение перебирает список автоматически
- **CORS** — браузер блокирует некоторые запросы к cobalt из-за политики CORS; handled через try/catch с перебором инстансов
- **Загрузка видео** — очень большие файлы (>2 ГБ) могут вызвать ошибку памяти при Blob-подходе; fallback — прямая ссылка
- **MD5** — реализован на чистом JS (не требует внешних библиотек), используется только для хэширования на стороне клиента
- **localStorage** — ограничен ~5 МБ; при переполнении запись будет молча провалена
- **Offline** — приложение работает offline, за исключением: загрузки шрифтов, QR-библиотеки и cobalt-загрузчика
